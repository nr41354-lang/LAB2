 ### 1. Задача 1
 Постановка задачи 
 
> Ваша задача – посмотреть, какие значения принимает последовательность при разных a0 и n, вывести закономерность, по которой строится последовательность и запрограммировать ее самостоятельно.

> Вводтится а0 и количество элементов последовательности. Проанализировав последовательность элементов от 1 до n, можно понять, что последовательность строиться следующим образом: берется текущее число и каждое его число умножаем на 2, затем получившиеся резутьтаты складыаем и получаем новый элемент последовательности. Но если а0 это 0, то и все следующие элементы тоже будут нулями.

 Входные данные 
 | Переменная | Тип          | min значение   | max значение  | Рекомендуемые значения min max|
 |------------|--------------|----------------|---------------|-------------------------------|
 |    a0      | целые числа  |10<sup>-9</sup> |10<sup>9</sup> |min  -1000  max 1000           |
 |    n       |целые числа   |10<sup>-9</sup> |10<sup>9</sup> |min 1     max 200              |

 Программа получает на вход целое числа а0 и n не превышающее 10<sup>9</sup> 
 |   | название переменной | Тип (в Java)  | 
 |---|---------------------|---------------|
 | а0| `а0`                | `int`         |
 | n | `n`                 | `int`         | 

 Алгоритм программы 
 1 Вводим данные: 
> Программа считывае целые числа а0 и n
2  Вычисление последовательнотии: 
> Сперва берем модуль а0(тк а0 может быть отрицательным, а последовательность состоит из положительных чисел). Затем берем последнюю цифру числа и умножаем ее на 2, далее уменьшаем число, отсекая последнюю цифру и также берем цифру, которая теперь стоит в конц числа и умножаем ее на 2, и т.д. пока число не станет равноым нулю, а затем складываем получившиеся значения и получаем новый элемент, с которым проделываем все тоже самое и так n раз. Если а0=0, то последовательность состоит из нулей.
 3 выводим получившуюся последовательность


 Программа
 ```java
import java.util.Scanner;
public class Main {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        System.out.print("a0: ");
        int a0 = in.nextInt();
        System.out.print("n: ");
        int n = in.nextInt();
        for (int i = 0; i < n; i++) {
            int next = 0;
            int c = Math.abs(a0);
            while (c != 0) {
                int d = c % 10;
                next += d * 2;
                c /= 10;
            }
            if (a0 == 0) {
                next = 0;
            }
            a0=next;
            System.out.print(a0+" ");
        }
    }
}
```

    ### 2. Задача 2

 Постановка задачи 
> С клавиатуры вводится натуральное число N (N < 20). Вывести на экран десятичное представление числа, которое в троичном виде состоит из N единиц.
> На вход подается число N, которое в троичном виде состоит из N едениц. Чтобы перевести число из троичной системы в десятеричную необходимо последнюю цифру числа умножить на 3<sup>0</sup>, предпоследнюю на 3<sup>1</sup>, ..., первую цифру на 3<sup>N-1</sup> и полученные результаты сложить.

 Входные данные 
 | Переменная | Тип          | min значение | max значение  |
 |------------|--------------|--------------|---------------|
 | N          | Целые числа  | 1            | 19            |

    
 Программа получает на вход челое число N
 |   | название переменной | Тип (в Java)  | 
 |---|---------------------|---------------|
 | N | `N`                 | `int`         |


 Алгоритм 
 1 Вводим данные: 
> Программа считывае целое число N
 2  Вычисление числа в десятичном представлении: 
 > Число N в троичном представлении состоит из N еденц, чтобы перейти в десятичное представление мы умножаем последнюю цифру на 3<sup>0</sup> и т.д., затем складывем получившиеся значения. Пусть h это десятичное представление числа N т пка оно рано нулю. Теперь начнем переводить число в десятичное представление: первое слагаемое всегда 1, далее мы 1 умножаем на 3 и прибавляем к 1, получившийся результат складыаем с 1*3<sup>2</sup> и т.д. 
 
 3 выводим получившийся результат

Программа 
 ```java
import java.util.Scanner;
public class Main {
    public static void main(String[] args) {
        Scanner in= new Scanner(System.in);
        System.out.print("N: ");
        int N = in.nextInt();
        long h =0;
        int p=1;
        for(int i=0; i<N; i++){
            h=h+1*p;
            p=p*3;
        }
        System.out.print(h);
    }
}
```

  ### 3. Задача 3
  
 Постановка задачи 
> Микрорайон с начальной численностью населения P0 развивается по годам: каждый год к текущему населению прибавляется прирост, равный целой части выражения P * g / 100, после чего учитывается миграция m. Смоделируйте развитие до момента завершения процесса. Выведите: (1) сколько полных лет прошло к моменту завершения процесса; (2) итоговую численность населения (если она превысила Cap, вывести ровно Cap).
> В программе необходимо посчитать количество лет за которые микрорайон завершит свое развитие. И также посчитать итоговую численность населения в микрорайоне, используя заданную, причем если она превышает Cap, то вывести Cap.

  Входные данные 
 | Переменная | Тип          | min значение | max значение  |
 |------------|--------------|--------------|---------------|
 |    P0      | Целые числа  | 1            | 10<sup>9</sup>|
 |    g       |  Целые числа | 1            | 100           |
 |    m       |  Целые числа | 1            | 100           |
 |    Cap     |  Целые числа | 1            | 10<sup>9</sup>|

Программа получает на вход целые числа 
 |   | название переменной | Тип (в Java)  | 
 |---|---------------------|---------------|
 | P0| `P0`                | `long`        |
 | g | `g`                 |`long`         |
 | m | `m`                 |`long`         |
 |Cap| `Cap`               | `long`        |


Алгоритм 
1 Вводим данные: 
> Программа считывае целые числа P0 g m Cap
 2  Вычисление итоговой численности населения и количество лет: 
>Пока количество населения не превышает вместимость численноть с каждым годом будет расти на m+ P0 * g / 100. как только P0 станет больше Cap выходим из цикла. Если P0 <= Cap, то P0=Cap. С каждым увеличением P0 счетчик лет увеличивается.
 
 3 выводим получившийся результат
 >выводится количество потраченного времени и итоговая численность населения.

 Программа 
  ```java
import java.util.Scanner;
public class Main {
    public static void main(String[] args) {
        Scanner in =new Scanner (System.in);
        System.out.println("начальная числ. нас-я, темп прироста нас-я, миграция, вместимость микр-а");
        long P0= in.nextLong(), g= in.nextLong(), m= in.nextLong(), Cap= in.nextLong();
        long y= 0;
        while (P0<Cap) {
            long k = P0 * g / 100;
            P0 += (m + k);
            y++;
        }
            if(P0>=Cap){
                P0=Cap;
            }
            System.out.print(y);
            System.out.print(P0);
    }
}
```

    ### 4. Задача 4

  Постановка задачи 
 Дана последовательность из n элементов. Известно, что все ее элементы находятся в диапазоне от 10 до 100 (включая границы). Определить, есть ли такие элементы, которые встречаются в последовательности ровно столько раз, каково их значение. Если такие элементы есть – вывести их значения, если их нет – вывести "NO".

необходмо посчитать сколько раз каждый элемен встречается в последовательности, и если значение элемента совпадает с его количеством то выводим это значение, а если же таких элементов нет то выводим NO. 

  Входные данные 
 | Переменная | Тип          | min значение | max значение  |
 |------------|--------------|--------------|---------------|
 |     n      | Целые числа  | 1            | 10<sup>9</sup>|

Программа получает на вход целые числа 
 |   | название переменной | Тип (в Java)  | 
 |---|---------------------|---------------|
